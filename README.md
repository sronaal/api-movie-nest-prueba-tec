# Prueba Tecnica NextJS Y NestJS

  ## Dependecias Proyecto
  1. TypeORM  (Modelar Tablas)
  2. mysql2   (Conector para la base de datos)
  3. @nestjs/typeorm ()


# Ejecucion de Proyecto
  ## Con Docker

  1. Clonar repositorio
  ```
  git clone <url>
  cd <carpeta>
  ```
  2. Crear Contenedores y ejecutar
  ```
  docker-compose up -d 
  ```
